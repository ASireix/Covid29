INFORMATIONS

À gauche, vous pouvez voir sur la carte différentes places sur lesquelles effectuer un test Covid.

À droite, différents onglets si vous desirez :

  • Les renseignements

  • Un formulaire pour donner votre avis sur l'établissement

      • Vous devrez y indiquer :

    • Un indentifiant, votre âge, votre sexe, la date du Test et celui du Résultat, et le Type du test
    • Ainsi qu'une note en selectionnant une étoile (1 = pas du tout satisfait 5 = très satisfait)

      • Vous pouvez :

    • Laisser un commentaire au sujet de l'établissement

  • Les graphiques des différents avis sur l'établissement
    • Nombre de tests répertoriés par mois
    • Évaluations du temps passé avant la réception des résultats
    • Évaluations de la qualitée du service
    • Jours écoulés avant le retour des tests
    • Âges des personnes ayant rempli le formulaire

  • Les mentions légales

AIDES

Carte

    Les bouttons + et - permettent de zoomer ou dézoomer

    Le nombre indique la quantite de site disponible dans la région

    Lorsqu'un virus apparaît, c'est un site unique. Cliquer dessus permet d'accéder à ses informations

Formulaire

    Tous les champs sont obligatoires, sauf l'espace commentaire

    Le site se selectionne en cliquant sur la carte

Statistique

    Les données affichées dépendent de l’établissement sélectionné sur la carte

    Les graphiques sont directement actualisées après avoir entré de nouvelles informations dans le formulaire
